"Send Mail"

URL:

N/A

Triggers:

Hover and Click

Rules:

When the user hovers over the "send mail" button the rectangle around the text changes shape and morphs into an ovular shape. When the user clicks on the "send mail" button the mail icon bounces then shifts into a paper airplane that flys away indicating that the mail is sent. Lastly, after the airplane animation is a complete a "reset" button similar to the "send mail" button fades in. That button also shifts on hover as well, and when clicked brings the user back to the original screen.
